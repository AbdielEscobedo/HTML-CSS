Introduction to HTML

HTML (HyperText Markup Language) is the standard language for creating web pages. It's the skeleton of any website, defining its structure and content. Along with CSS (for styling) and JavaScript (for interactivity), it forms the foundation of web development.

HTML uses tags to structure content. For example, a paragraph is defined by the `<p>` tag, and a main heading by `<h1>`.

Importance and Evolution

  • It's the essential first step in learning web development.
  • HTML5, the latest major version, introduced semantic tags and multimedia support.
Basic HTML Document Structure

Every HTML document follows a fundamental structure to be correctly interpreted by browsers.


            <!DOCTYPE html>
            <html lang="en">
            <head>
            <meta charset="UTF-8">
              <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Document Title</title>
            </head>
            <body>
            <!-- Page content goes here -->
             </body>
            </html>

Key components:

  • `<!DOCTYPE html>`: Document type declaration (HTML5).
  • `<html>`: The root element, often with a `lang` attribute.
  • `<head>`: Contains metadata (not displayed on page).
  • `<body>`: Contains all visible page content.
Elements and Attributes

In HTML, content is organized into **elements**. An element is defined by an opening tag, content, and a closing tag (e.g., `<p>Hello</p>`).

Attributes provide additional information about an element and are placed within the opening tag (e.g., `<a href="link.html">`).


<!-- A paragraph element -->
<p>This is a paragraph.</p>

<!-- An image element with src and alt attributes -->
<img src="image.jpg" alt="A description of the image">

Some elements are "empty" and do not require a closing tag, like `<br>` (line break) or `<img>`.

Core HTML Tags

Here are some of the most common and essential HTML tags:

  • `<h1>` to `<h6>`: Headings for different levels of importance.
  • `<p>`: Defines a paragraph of text.
  • `<a>`: Creates a hyperlink to another page or resource.
  • `<img>`: Embeds an image into the document.
  • `<ul>`, `<ol>`, `<li>`: Used for unordered, ordered, and list items respectively.

<h2>Section Title</h2>
<p>This is some content.</p>
<a href="https://example.com">Visit Example</a>
Text Formatting

HTML provides various tags to format text, enhancing its meaning and presentation.

Use `<strong>` or `<b>` for **bold text**, and `<em>` or `<i>` for *italic text*.


<p>This is <strong>important</strong> text.</p>
<p>A phrase with <em>emphasis</em>.</p>

Other tags include `<br>` for a line break, `<hr>` for a horizontal rule, and `<blockquote>` for block quotes.

Links Hyperlinks

This section will explain the `<a>` (anchor) element, the cornerstone of web navigation, and its essential attributes.

The `<a>` (Anchor) Element

The `<a>` tag is used to create hyperlinks, which connect one web page to another, or one section of a page to another section. It transforms text or images into clickable navigation elements.

Basic Structure:


<a href="destination-url">Link Text</a>

The href attribute is the most crucial attribute for the `<a>` tag. It specifies the URL (Uniform Resource Locator) that the hyperlink points to. This URL can be:

  • Absolute URL: A full web address (e.g., https://www.google.com). Used for linking to external websites.
  • Relative URL: A path to another file or resource within the same website (e.g., about.html, images/picture.jpg). Used for internal navigation.
  • Anchor (Jump) Link: A link that points to a specific section within the same HTML document. This is achieved by using a # followed by the id of the target element (e.g., <a href="#section-id">Go to Section</a>).

Types of Links

HTML links serve different purposes:

  • External Links: Navigate to a different website or domain. 
    <a href="https://developer.mozilla.org">Visit MDN Web Docs</a>
  • Internal Links: Navigate to other pages or resources within the same website. 
    <a href="contact.html">Contact Us</a>
  • Anchor (Jump) Links: Allow users to jump to a specific part of the current page. The href must match an element's id. 
    <a href="#Introduction_to_HTML">Back to Introduction</a>

Useful Attributes for `<a>`

Other important attributes enhance link functionality:

  • target attribute: Specifies where to open the linked document. 
    <a href="https://newsite.com" target="_blank">Open in New Tab</a>
    _blank opens in a new tab; _self (default) opens in the same tab.
  • title attribute: Provides extra information (tooltip) when hovering over the link. 
    <a href="products.html" title="View our product catalog">Our Products</a>
  • rel attribute: Defines the relationship between the documents (e.g., nofollow, noopener). 
    <a href="untrusted-link.html" target="_blank" rel="noopener noreferrer">External Link Securely</a>

Links to Specific Resources

Beyond web pages, links can trigger other actions:

  • Email Links (mailto:): Opens the user's default email client. 
    <a href="mailto:info@example.com?subject=Inquiry">Send us an Email</a>
  • Telephone Links (tel:): Allows direct calls on mobile devices. 
    <a href="tel:+15551234567">Call Us</a>

Best Practices for Links

  • Descriptive Link Text: Use meaningful text (e.g., "Learn More About HTML") instead of generic "Click Here." This aids accessibility and SEO.
  • Accessibility: Ensure links are clearly distinguishable from regular text.
  • Broken Links: Regularly check and fix any broken links to maintain a good user experience.
